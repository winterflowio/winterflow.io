  • When reducing the screen in the order page, the icon of error, confirmation, warning messages is broken

  • The scrollbar is missing in the new multistore dropdown

  • The summary and description fields of the Product Page form allow more characters than the defined limit
  • When running a CLI command prestashop:update:sql-upgrade-file-hooks-listing and prestashop:update:configuration-file-hooks-listing, an error is displayed
  • The new Multistore header is not displayed on Product Add / Edit page
  • When modifying a translation of Email Body, an error is displayed
  • When searching for a translation through the interface, the « leaves » of the tree that do not include the searched term are not disabled
