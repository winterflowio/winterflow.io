How it works

A lightweight agent that turns any server into your private cloud.

Step 1

Prepare a Server

Use any cloud provider, VPS, or even a home server.

Step 2

Install Agent

Set up the Winterflow agent. Root access is only needed for installation, not for running the agent.

Step 3

Secure Connection

The agent generates a key pair and connects to Winterflow using mutual TLS (mTLS).

Step 4

Zero-Trust Secrets

Sensitive data is encrypted in the browser with the agent's public key and decrypted only on your server.
